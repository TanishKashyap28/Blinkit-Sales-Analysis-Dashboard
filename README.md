# Blinkit Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit sales data and provides insights into sales performance, customer behavior, and product trends.

## Objectives
- Analyze overall sales performance
- Track product category performance
- Monitor customer purchasing patterns
- Identify top-performing products and outlets

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
- DAX

## Dashboard Features
- Total Sales Analysis
- Product Category Insights
- Outlet Performance Tracking
- Interactive Filters and Slicers
- KPI Cards and Visualizations

## Key Insights
- Identified top-selling product categories
- Analyzed sales trends across outlets
- Compared performance of different product types
- Provided business insights for decision-making
